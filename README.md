# Long Novelist — 百万字长篇小说创作助手

一个为百万字级长篇小说创作而设计的 Claude Code Skill。

## 核心功能

- **自主配置**：13步交互式问答，自主控制题材、每章字数、写作风格、模仿对象（联网搜索）
- **大纲系统**：整体大纲（起承转合四段）→ 细纲（每章核心事件+章首引子+结尾钩子+伏笔操作）
- **角色管理**：JSON角色档案（性格/状态/当前位置/关系网/状态变更日志），每章自动更新
- **伏笔追踪**：JSON伏笔清单（铺设/回收状态管理），分散铺设集中回收
- **交互写作**：用户驱动节奏，每章写完后展示并等待反馈，支持修改迭代
- **跨会话恢复**：剧情梗概+写作计划+角色档案确保新会话快速恢复上下文
- **用户偏好记忆**：自动记录写作偏好和反馈模式，跨会话个性化体验
- **大纲辅助**：卡文时联网搜索提供创作建议
