# 言情甜宠桥段素材

> 言情线核心公式：甜宠不是一直发糖，而是酸涩中的甜更动人。虐恋不是一直虐，而是虐到极点后的救赎最动人。

---

## 一、追妻火葬场七步法

```
甜蜜期 → 误会/伤害 → 分离/失去 → 追悔+成长 → 追回行动 → 反复考验 → 最终和解
```

| 步骤 | 章节占比 | 核心任务 | 读者情绪 |
|------|----------|----------|----------|
| 1. 甜蜜期 | 10% | 建立感情基础，让读者"入股"这对CP | 甜、期待 |
| 2. 误会/伤害 | 10% | 男主因傲慢/偏见/秘密伤害女主 | 愤怒、心疼女主 |
| 3. 分离/失去 | 10% | 女主离开/消失，男主意识到失去 | 爽（女主硬气）+ 期待 |
| 4. 追悔+成长 | 20% | 男主反思错误，改变自己 | 酸涩、开始原谅 |
| 5. 追回行动 | 25% | 男主用行动证明改变，重新追求 | 甜中带酸、期待 |
| 6. 反复考验 | 15% | 女主不轻易原谅，设置考验 | 紧张、怕再碎 |
| 7. 最终和解 | 10% | 彻底和解，感情升华 | 圆满、满足 |

### 关键操作要点

- **伤害必须有原因**：不能无缘无故渣，必须有让读者"虽恨但理解"的动机
- **追妻必须真付出**：不能只是嘴上说"我错了"，必须有实质性改变和牺牲
- **女主必须有骨气**：被伤害后不能轻易原谅，设置合理的考验期
- **节奏口诀**：甜三章、虐两章、追五章、和一章

## 二、暧昧期十大节点

暧昧是言情小说最重要的阶段——"确定关系前的每一次靠近，都比在一起后的吻更让人心动。"

| 节点 | 场景类型 | 经典写法 |
|------|----------|----------|
| 1. 初见 | 意外/冲突/惊艳 | 不是一见钟情，而是一见"在意"——这个人跟别人不一样 |
| 2. 第二次相遇 | 巧合/被迫 | 发现对方不为人知的一面（脆弱/温柔/强硬） |
| 3. 被迫独处 | 困在一起 | 电梯故障、被困雨夜、出差同住——空间压缩制造亲密 |
| 4. 第一次触碰 | 无意的肢体接触 | 手指碰到手指，同时缩回，空气凝固三秒 |
| 5. 吃醋预警 | 第三方出现 | 看到对方和异性在一起，心里不舒服但嘴上不承认 |
| 6. 英雄救美 | 保护/挺身而出 | 不是玛丽苏式拯救，而是"在场"——TA最需要的时候在场 |
| 7. 深夜联系 | 打破边界 | 深夜发消息"睡了吗"，聊天到凌晨——白天理性和夜晚感性的博弈 |
| 8. 秘密共享 | 信任升级 | 倾诉从未告诉别人的事，建立"我们之间的秘密" |
| 9. 临界告白 | 差点说出口 | "我有个事想告诉你……算了，下次再说" |
| 10. 正式告白 | 情感爆发 | 不是简单"我喜欢你"，而是带着情节推动和情感重量的告白 |

## 三、告白桥段库

### 经典告白类型

| 类型 | 写法 | 适用场景 |
|------|------|----------|
| 隐晦告白 | "以后的每一个冬天，我都想和你一起过。" | 暧昧期后期，性格内敛的角色 |
| 危机告白 | "如果你活下来，我就娶你。" / 受伤昏迷时的真心话 | 生死关头，情感爆发 |
| 反向告白 | "你听好了，我不是在征求你的同意，我是在通知你。" | 霸总/强势角色 |
| 遗憾告白 | "当初没说出口的话，现在说还来得及吗？" | 重逢/破镜重圆 |
| 契约告白 | "我们结婚吧。不是因为爱，是因为我需要一个理由留下来。" | 先婚后爱/契约恋爱 |
| 醉酒告白 | 喝了酒说真话，第二天"我昨天说了什么？" | 轻松甜宠 |
| 信息告白 | 一条编辑了又删、删了又编辑的消息，最后只发了两个字 | 现代都市 |

### 告白写作三要素

1. **场景铺垫**：告白不是发生在真空里——雨夜、黄昏、初雪、机场、医院走廊
2. **情绪铺垫**：告白前的沉默/深呼吸/握紧的手，比告白本身更重要
3. **回应设计**：不是简单的"好"——可以是沉默三秒后的一个拥抱、一句"你怎么才说"

## 四、甜宠黄金公式

### 日常甜宠十连招

| 编号 | 招式 | 示例 |
|------|------|------|
| 1 | 摸头杀 | 不经意的揉头发，眼神温柔 |
| 2 | 壁咚 | 空间压缩制造心跳加速 |
| 3 | 投喂 | 记住TA爱吃的，随时准备好 |
| 4 | 披外套 | 冷的时候自然地脱下外套披上 |
| 5 | 挡酒 | "她喝不了，我替她。" |
| 6 | 睡颜杀 | 偷看TA睡着的样子，嘴角不自觉上扬 |
| 7 | 背后抱 | 从背后环住，下巴抵在肩膀上 |
| 8 | 额头吻 | 比嘴唇更克制的深情 |
| 9 | 专属昵称 | 只有两个人懂的称呼 |
| 10 | 第一反应 | 遇到危险/好消息时第一个想到的是TA |

### 甜宠节奏控制

```
高甜场景不能连续超过3章 → 否则读者会腻
正确节奏：甜2章 → 小波折1章 → 甜3章 → 中波折2章 → 甜1章(高甜) → 大高潮
```

## 五、虐恋节奏公式

### 虐的层次递进

| 层级 | 类型 | 效果 | 示例 |
|------|------|------|------|
| 轻虐 | 误会/错过 | 读者心里发酸 | 两个人明明相爱却说反话 |
| 中虐 | 被迫分离 | 读者鼻酸/眼眶红 | 家族阻挠/身世秘密/绝症误会 |
| 重虐 | 生死两隔 | 读者落泪 | 为救对方牺牲自己 |
| 终极虐 | 双向奔赴却永不能在一起 | 读者哭崩 | 命运不可抗力+巨大牺牲 |

### 虐后必甜法则

> 虐一章，甜三章。虐到极点的章节之后，必须给读者情感出口。

## 八、避坑指南

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 女主傻白甜到让人生气 | 女主可以单纯但不能蠢，必须有成长线 |
| 男主莫名其妙就情深似海 | 感情必须有铺垫，每一次靠近都要有理由 |
| 误会全靠不张嘴解释 | 可以"解释了但对方不信"，不能"就是不解释" |
| 女二纯粹是使绊工具人 | 女二也应该有动机和逻辑，有独立人格 |
| 甜宠全程无冲突 | 即使是甜宠文，也需要外部压力制造悬念 |
| 虐恋全程虐到底 | 虐恋必须在绝望中给希望，压抑中给出口 |
| 感情进度条停滞 | 每3章必须有感情线推进——或近或远，但不能停 |

## 九、2025言情微创新

1. **女主人设升级**：从"被保护"到"互相保护"，女主有能力有事业有独立思考
2. **去套路化告白**：拒绝中规中矩，更生活化、更有角色个性的告白方式
3. **职业线融合**：感情线与职业成长线并进，两条线互相推动
4. **配角感情线**：给重要配角完整的感情弧线，避免工具人
5. **"甜"的定义扩展**：不只身体接触才甜——默契、信任、成长、互相成就，都是"甜"
