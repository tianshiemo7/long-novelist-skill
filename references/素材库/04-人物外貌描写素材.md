# 人物外貌描写素材

> 核心原则：不写"大眼睛高鼻梁"的流水账。锚定一个独有特征，用矛盾感塑造记忆点。

---

## 一、眼睛

### 眼型速查
丹凤眼、瑞凤眼、柳叶眼、桃花眼、月牙眼、细长眼、杏眼、三角眼

### 眼神/目光
| 类型 | 词汇 |
|------|------|
| 正面 | 清澈、明亮、深邃、坚定、炯炯有神、灿若星辰、含情脉脉、顾盼生辉 |
| 负面 | 呆滞、浑浊、涣散、阴鸷、晦暗、空洞 |
| 锐利型 | 犀利如刃、两道光射寒星、锋芒毕露 |
| 温柔型 | 眸光潋滟如浸酒、双目似浸在水里的琥珀、眼波流转 |

### 不落俗套的进阶描写
- **灵动型**：瞳仁盛着碎星 / 眼睛忽闪忽闪像会说话 / 眨巴眨巴的
- **破碎型**：眼底碎了一汪星子 / 眼窝深处盛着未干涸的海
- **冷峻型**：眼尾挑起的讥诮 / 目如寒潭 / 眼睛里结着冰碴
- **沧桑型**：眼睛像两颗琥珀藏着星辰与风霜 / 目光深处有一层挥之不去的灰
- **矛盾型**：看似温柔的眼眸深处藏着深不见底的忧郁 / 嘴角挂着笑意眼底却冷

## 二、眉

### 眉型速查
剑眉、柳叶眉、一字眉、远山眉、黛眉、浓眉、淡眉、粗眉、细眉

### 写法
- 剑眉斜插入鬓 — 英武之气
- 眉如远山含黛 — 清秀温婉
- 眉峰压着山影 — 郁结/沉重
- 眉头拧成一个川字 — 忧愁/思考
- 眉梢微挑 — 挑衅/质疑/兴趣

---

## 三、发型

### 发色发质
乌黑如瀑 / 银白如霜 / 青丝如云 / 华发斑白 / 如绸缎般柔顺 / 蓬松卷曲慵懒

### 不落俗套写法
- **光影感**：碎发被日光镀成琥珀色 / 银发泛着淡淡的月光仿佛月光凝成
- **氛围感**：发梢洇着暮色 / 海藻般浓密长发慵懒地垂在肩头
- **特殊设定**：一头妖冶的紫发 / 发丝如雪透着超凡脱俗的美
- **神秘感**：发丝如黑色的云遮住半张脸

---

## 四、面容/五官

### 古风
- 面如冠玉 / 面若桃瓣 / 肤如凝脂 / 颜如渥丹
- 冰肌藏玉骨 / 丹脸赛胭脂 / 眉目如画

### 现代
- 轮廓分明 / 下颌线精致锋利 / 鼻梁高耸有雕塑感
- 皮肤白皙透着淡淡红晕 / 小麦色健康肤色

### 不落俗套
- **特征锚定**：左耳的痣上长着根细毛 / 笑时露出虎牙撞出小小缺口
- **身份暗示**：指节上泛着薄茧—暗示工匠/乐手/剑客
- **反差感**：面容憔悴却依旧保持一丝尊严 / 满脸风尘掩不住那份倔强

## 五、身材/身姿

### 女性
| 类型 | 词汇 |
|------|------|
| 纤瘦 | 苗条、亭亭玉立、杨柳细腰、纤腰一握、身姿婀娜 |
| 高挑 | 身材高挑如模特、双腿修长 |
| 丰满 | 丰腴匀称、曲线玲珑 |

### 男性
| 类型 | 词汇 |
|------|------|
| 健壮 | 魁梧、虎背熊腰、身强力壮、肌腱发达、肌肉线条利落 |
| 文雅 | 身如玉树、长身玉立、玉树临风、飘若游云矫若惊龙 |
| 精瘦 | 瘦而有肉筋骨分明、身形修长露出两根秀气的锁骨 |

### 不落俗套写法
- **脊背挺成松枝的骨** — 刚直坚毅
- **背影瘦成一竿竹，暮色里站成了蒹葭** — 孤独+风骨
- **身形笔挺，端端正正坐出了一种自律感** — 通过姿态传达性格
- **他的领带永远泛着冷铁的光，侧影如黑檀木浮雕** — 现代冷峻型

## 六、气质

### 古典气质速查
| 类型 | 表达 |
|------|------|
| 温润儒雅 | 温润如玉、谦谦君子、立如芝兰玉树笑如朗月入怀 |
| 清冷出尘 | 清癯出尘、湛然若神、秋水为神玉做骨 |
| 英武不凡 | 龙章凤姿、气宇轩昂、一双眼光射寒星 |
| 洒脱不羁 | 狂傲不羁、萧萧肃肃、爽朗清举 |
| 深沉内敛 | 沉敛如陈年酿、和光同尘、肃肃如松下风 |

### 矛盾张力式（最出彩）
> - 她的笑靥比糖纸还甜，眼底却结着薄冰
> - 递过来的茶盏烫得他指尖发颤——那层冰下面，是烧得滚烫的火
> - 看似温柔的眼眸深处藏着深不见底的忧郁
> - 悲悯藏在骨血里，眼神却锐利如刀

## 七、外貌描写四技法

### 1. 细节锚定法
不写通用模板，捕捉独一无二的个人标志：
> "他左耳的痣上长着根细毛，风一吹像小旗子似的抖"

### 2. 感官联动法
调动视觉以外的感官：
> "他的笑声像碎冰撞在银盘上，溅起的凉意顺着脖颈爬进衣领"

### 3. 矛盾张力法
用反差塑造复杂性格：
> "手指白净修长如弹钢琴的手，掌心却满是老茧"

### 4. 文化意象法
嫁接古典诗词与现代描写：
> "背影瘦成一竿竹，风里站成《诗经》里的蒹葭"
