# 多题材专题素材

> 不同题材有不同的创作规律和爽点引擎。本文件覆盖悬疑推理、历史古风、科幻末世、无限流四大题材的核心技法。

---

## 第一篇：悬疑推理

---

### 一、悬疑推理核心公式

```
悬疑 = 谜题设置 × 线索密度 × 反转力度
```

读者读悬疑的心理：我要比主角先猜到真相——但不要让我真的猜到。

---

### 二、线索设计系统

#### 线索投放节奏

| 章节阶段 | 线索类型 | 投放量 | 作用 |
|----------|----------|--------|------|
| 前10% | 红鲱鱼（假线索） | 2-3条 | 引导读者往错误方向走 |
| 15%-40% | 真实线索（碎片） | 每3章1条 | 让细心的读者开始怀疑 |
| 40%-60% | 关键线索（转折） | 1-2条 | 推翻初步推理，打开新方向 |
| 60%-85% | 确认线索（收束） | 逐步增加 | 真相慢慢浮出水面 |
| 85%-100% | 决定性证据 | 1条 | 锁定真凶/真相 |

#### 线索的三种埋法

| 方法 | 操作 | 示例 |
|------|------|------|
| 明埋 | 写出来但不强调，夹在无关信息中 | 现场描述中"茶几上有半杯凉掉的咖啡"——真正重要的是咖啡杯上的口红印颜色 |
| 暗埋 | 只写现象不写解释 | "他看到书房窗户开了条缝，皱了皱眉。"（不做评价，后文揭示重要性）|
| 侧埋 | 通过对话/回忆不经意提及 | "说起来，那天停电的时候，只有地下室的灯还亮着，真奇怪。" |

---

### 三、反转设计

#### 反转强度等级

| 等级 | 类型 | 效果 | 使用位置 |
|------|------|------|----------|
| Lv1 | 身份反转 | "原来TA是卧底" | 中段高潮 |
| Lv2 | 动机反转 | "凶手不是为了钱" | 中后段 |
| Lv3 | 时间线反转 | "事件不是那天发生的" | 后段 |
| Lv4 | 叙事者反转 | "叙述者本人就是凶手/死者" | 结尾 |
| Lv5 | 世界观反转 | "这不是现实世界" | 终极结局 |

#### 反转三原则

1. **回看合理**：反转后读者回看前文，每个细节都说得通
2. **铺垫在前**：反转前必须埋至少3个相关线索（太隐晦也不行）
3. **不过度反转**：一部作品1-2次重大反转足够，过多会审美疲劳

---

### 四、经典推理套路

| 套路 | 核心设定 | 推荐长度 |
|------|----------|----------|
| 暴风雪山庄 | 封闭空间连环死亡，凶手在众人之中 | 中篇(5-10万字) |
| 倒叙推理 | 先告诉凶手是谁，再讲如何破案 | 中篇 |
| 叙述性诡计 | 利用叙述视角和信息差欺骗读者 | 长篇(15-30万字) |
| 交换杀人 | A杀B想杀的人，B杀A想杀的人 | 中长篇 |
| 不在场证明破解 | 核心谜题是时间/空间的不可破解性 | 短中篇 |
| 连续杀人模式 | 连环案件的模式识别与破解 | 长篇 |

---

### 五、推理节奏控制

```
发现线索（好奇）→ 线索中断（焦虑）→ 新方向（希望）→ 再次落空（绝望）→ 灵光一现（突破）
```

**关键**：每次希望都比前一次更接近真相，每次绝望都比前一次更沉重。

## 第二篇：历史古风

---

### 六、历史权谋核心法则

#### 权谋三要素

| 要素 | 说明 | 写作要点 |
|------|------|----------|
| 信息差 | 我知道你不知道 | 这是权谋的本质——谁掌握更多信息谁赢 |
| 资源差 | 我有你没有 | 兵力/财力/人脉/把柄 |
| 时间差 | 我先你一步 | 提前布局，等对手落子时已输 |

#### 权谋博弈五层

```
第一层：简单的计谋（调虎离山/声东击西）
第二层：对手看穿了第一层（将计就计）
第三层：你料到对手会看穿（计中计）
第四层：对手料到你会料到（套中套）
第五层：所有人都错了——真正的目的从来不是这个
```

**写作原则**：一般写到第三层即可，关键局写到第四层。第五层留作全书最大高潮。

### 七、朝堂/宫斗场景要素

| 要素 | 操作 |
|------|------|
| 一句话的杀伤力 | 朝堂上最狠的杀招不是长篇奏疏，而是一句看似无心的话——"臣听闻……" |
| 沉默的力量 | 群臣沉默、皇帝沉默、主角沉默——谁先开口谁就输了 |
| 送礼的政治 | 送了什么东西、什么时候送、由谁送——全部是政治信号 |
| 站位学 | 朝堂上站的位置、后宫坐在谁的旁边——都是势力版图的缩影 |
| 缺席的存在感 | 谁没来参会、谁被排除在宴请名单外——不在场一样能传递信息 |

### 八、历史细节真实感速成

#### 五感还原法

| 感官 | 细节方向 | 示例 |
|------|----------|------|
| 视觉 | 光线来源（烛/油灯/火把/月光）| "烛火在他脸上跳了跳，影子在墙壁上拉得很长" |
| 听觉 | 环境音（更鼓/马蹄/梆子/虫鸣）| "远处传来三更的梆子声，一声比一声远" |
| 嗅觉 | 时代气味（烛油/墨香/药味/香料/潮湿的木头）| "屋子里是旧书和中药混合的味道" |
| 触觉 | 材质触感（粗糙的纸/冰凉的玉/温润的瓷）| "竹简的绳子已经松了，一碰就散" |
| 味觉 | 食物/茶酒 | "茶已经凉了，涩味在舌根久久不散" |

## 第三篇：科幻末世

---

### 九、末世世界观搭建

#### 末世类型

| 类型 | 核心设定 | 核心冲突 |
|------|----------|----------|
| 丧尸末世 | 病毒感染，文明崩塌 | 活着 vs 变异，人性 vs 兽性 |
| 核战后 | 辐射废土，资源稀缺 | 生存 vs 道德，部落 vs 部落 |
| 人工智能 | AI觉醒/反叛 | 人类 vs 造物主，自由 vs 安全 |
| 外星入侵 | 外族降临 | 地球文明 vs 外星文明 |
| 环境崩溃 | 气候灾难/资源枯竭 | 人类 vs 自然，秩序 vs 混乱 |
| 赛博朋克 | 高科技低生活 | 个体 vs 系统，肉体 vs 机械 |

#### 末世铁律

1. **资源永远不够**——每次得到资源都必须付出代价
2. **信任是最稀缺品**——不要让人物轻易相信陌生人
3. **安全区不存在**——任何看起来安全的地方都会沦陷
4. **规则被重写**——旧世界的道德不再适用，新世界有新的规则

### 十一、末世生存法则

#### 资源等级

| 优先级 | 资源 | 争夺烈度 |
|--------|------|----------|
| S级 | 净水/药品/能源 | 最高——不惜一切代价 |
| A级 | 食物/弹药/庇护所 | 高——但愿意交易 |
| B级 | 衣物/工具/燃料 | 中——按需分配 |
| C级 | 奢侈品/旧世界货币 | 低——只有基地稳定后才会有需求 |

#### 幸存者团体原型

| 类型 | 特征 | 剧情功能 |
|------|------|----------|
| 军阀型 | 武力至上，强者为王 | 反派/临时盟友 |
| 宗教型 | 以信仰凝聚人心 | 可疑的安全区/潜在威胁 |
| 科学家型 | 研究病毒/科技，寻找解药 | 希望所在/最脆弱也最重要 |
| 平民型 | 普通避难者，抱团取暖 | 主角保护对象/人性底线 |
| 掠夺者 | 不生产只掠夺 | 移动威胁/常规敌人 |

## 第四篇：无限流

---

### 十二、无限流核心架构

#### 什么是无限流

主角被拉入一个"系统空间"，必须在一个个副本世界中完成任务才能生存/变强/获得奖励。副本可以是电影、小说、游戏、历史事件或完全原创的世界。

#### 三大核心要素

| 要素 | 说明 |
|------|------|
| 系统/主神空间 | 发布任务、提供兑换、管理规则 |
| 副本世界 | 每个副本一个独立故事，有任务目标和死亡威胁 |
| 强化体系 | 完成任务获得积分，兑换能力/装备/血统 |

---

### 十三、副本设计公式

```
副本 = 世界观 × 核心规则 × 主线任务 × 隐藏要素
```

#### 副本类型

| 类型 | 特点 | 副本长度 |
|------|------|----------|
| 生存类 | 活过指定时间即为胜利 | 短(3-5章) |
| 解谜类 | 需要破解谜题/找出真相 | 中(5-8章) |
| 对抗类 | 团队对战或与副本Boss对战 | 中短(4-6章) |
| 竞速类 | 多个队伍抢一个目标 | 短(3-5章) |
| 逃杀类 | 限定人数，活到最后的胜出 | 中(5-10章) |
| 大型战役 | 多阶段综合副本 | 长(10-20章) |

#### 副本设计检查清单

- [ ] 核心规则是否清晰无歧义？
- [ ] 是否有利用规则的反套路空间？（这才是无限流的灵魂）
- [ ] 副本世界是否有独立的故事逻辑？
- [ ] 是否有隐藏任务/隐藏奖励？
- [ ] 通关方式是否≥2种？（暴力/智取/规则漏洞）
- [ ] 副本难度与主角当前实力是否匹配？

---

### 十四、规则杀与反套路

#### 规则设计原则

1. **规则必须绝对公平**——系统不会偏袒，但可以利用
2. **规则必有漏洞**——这是主角的胜机，但必须合理
3. **违规则必死**——建立规则的神圣性，让利用规则更爽

#### 反套路五法

| 方法 | 操作 | 读者爽点 |
|------|------|----------|
| 文字游戏 | 规则用词有歧义，主角抓住漏洞 | 智力碾压 |
| 信息垄断 | 主角比对手多知道一条隐藏规则 | 信息差优势 |
| 规则叠加 | 两条规则叠加产生意外的效果 | 创造力爽感 |
| 借刀杀人 | 利用规则让系统/环境消灭敌人 | 不用自己动手的爽 |
| 掀桌子 | 不遵守副本规则，重新定义"通关" | 颠覆性震撼 |

### 十五、无限流强化体系

#### 强化方向

| 方向 | 示例能力 | 进阶路径 |
|------|----------|----------|
| 肉身强化 | 体质/力量/速度/再生 | 普通人→超级士兵→超凡体质→不灭金身 |
| 精神强化 | 念力/精神力/心灵感应 | 敏感→念力移物→精神控制→心灵世界 |
| 血统强化 | 吸血鬼/狼人/神族/魔族 | 初级血统→血脉觉醒→王级→始祖 |
| 技能强化 | 格斗/枪械/魔法/炼金 | 入门→精通→大师→领域创造 |
| 装备强化 | 武器/防具/道具 | 普通→精良→史诗→传说→神器 |
| 知识强化 | 秘籍/卷轴/传承 | 理论→实践→精通→创新→开宗立派 |

#### 兑换定价参考

```
低级能力：100-1000积分
中级能力：1000-5000积分
高级能力：5000-20000积分
顶级能力：20000+积分
特殊/唯一能力：需要隐藏任务获取，不可直接兑换
```
