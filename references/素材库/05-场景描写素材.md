# 场景描写素材

> 核心技巧：不写"一个很大的房间"，用动作触发空间反馈。不写"风景很美"，用感官叠加制造氛围。

---

## 一、场景造景七步法

| 步骤 | 操作 | 示例（武侠对决场景） |
|------|------|---------------------|
| 1.铺大块 | 明确环境结构 | 山巅险峰平台，周围悬崖峭壁 |
| 2.上色调 | 定环境底色 | 夜晚，朦胧月色，偏暗偏灰 |
| 3.上光影 | 增加立体感 | 月光照松树轮廓，树下黑影重重 |
| 4.上动静 | 让环境鲜活 | 大风呜呜，松枝哗哗，阴影摇摆 |
| 5.上细节 | 突出重点 | 树干粗糙、怪石冰凉硌脚 |
| 6.上联想 | 延展想象 | "仿佛魔窟炼狱，又似阎罗殿" |
| 7.上视角 | 人物体验环境 | 手摸地面冰凉，雷声让人心慌 |

## 二、空间描写——动态压缩法

不写长宽高尺寸，让角色动作触发空间反馈：

> ❌ 死写法："书房长20米宽10米，东墙书柜放史书……"
> ✅ 动态法："后退时脊背撞上冰冷书柜，羽毛笔从书隙滑落，剑架嗡鸣震得窗框扑簌落灰"

**空间折叠公式**：角色动作触发的空间反馈 + 关键物品动态暴露

### 三个维度同时着墨
- **垂直锚点**：穹顶血藤垂落 / 地砖裂隙涌瘴气
- **水平锚点**：廊柱切割视线 / 回音死角骤停
- **标志物**：九龙柱爪痕深三寸，抓着一角浸血皇诏

## 三、场景类型素材

### 战斗场面
| 步骤 | 写法 |
|------|------|
| 铺垫 | 先写被摧毁之物的宏大——"远处那颗星辰诞生已几亿年，表面一片红黄废土……" |
| 转场 | 自然过渡——"然而星辰虽大，在主角的气场下也显得渺小" |
| 过程 | 侧面烘托——"星辰剧烈颤动，表面涌起狂风飞沙，龟裂纹路蔓延" |
| 收尾 | 连锁反应——"碎片化作星际尘埃，撞向行星带" |

**战争群像**：先用广角定调（火箭织成火网）→ 再听觉渲染（号角/喊杀/兵器碰撞）→ 最后落到小人物视角

### 城市/街道
- 写出独特元素：越南餐厅飘出河粉香、工地尘土、街角艺人的表演
- 反映时代/阶层：人行道拥挤还是空荡？建筑代表贫穷还是富裕？
- 利用节奏感：早晚高峰车流模式、店铺开张打烊的节奏

### 废墟/荒城
> "这座深陷地下的古城已长眠多年，水流从鹅卵石街道上流过，建筑倾倒在一起形成诡异的拱门，除了滴水声外四周寂静无声，空气里充满腐坏的气味……"

### 自然/田园
- **巨大化写作**：大数词+物理量词——"成千上万吨积雨云""数万公顷森林"，空镜中塞进小小的人
- **列锦法**：枚举名词增强真实感——"松毛榉、胡桃楸、三花槭、红皮云杉……"
- **天气映射情绪**：晴天配希望、雨天配忧伤、秋日配变故将至

## 四、空间情绪配方

| 情绪 | 垂直锚点 | 水平锚点 | 标志物隐喻 |
|------|----------|----------|------------|
| 压迫 | 穹顶垂落铁荆棘 | 廊柱如巨人肋笼 | 王座嵌逆鳞 |

| 悲壮 | 天幕低垂铅灰色云 | 断旗斜插焦土 | 断剑半截入石 |

## 五、宫殿/权力空间

### 写作要点
- 选择**权力符号**作为标志物（玉玺、血诏、龙椅）
- 用**材质异常+能量外溢**营造奇幻感
- 每章只投放1-2颗地标炸弹，防止记忆过载
- 借助人群作为参照物体现宏大

### 示例：青铜门蚀刻婴儿哭脸纹，门缝钻出霜色触手。九龙柱爪痕深三寸，抓着一角浸血皇诏。

---

## 六、五感调度

场景描写不需要五感全开，精选1-2个即可。对于氛围类场景，优先设计**光影+声音**。

| 感官 | 适用场景 | 常用元素 |
|------|---------|---------|
| 视觉 | 所有 | 颜色、光影、明暗、形状、动态 |
| 听觉 | 所有 | 人声、环境音、音乐、沉默 |
| 触觉 | 动作/亲密/恐惧 | 温度、质感、疼痛、压力、潮湿 |
| 嗅觉 | 封闭空间/食物/恐惧 | 香气、体味、腐败、金属味 |
| 味觉 | 进食/受伤/亲密 | 食物味、血腥、苦涩 |
