## 一、经典修仙境界体系

### 主流体系（炼气→飞升）
| 大境界 | 细分 |
|--------|------|
| 炼气期 | 1-9层，引气入体，改变体质 |
| 筑基期 | 初/中/后/圆满，真气化液，打通经脉 |
| 金丹期 | 初/中/后/圆满，凝结金丹，寿逾千年 |
| 元婴期 | 初/中/后/圆满，金丹化婴，元神初成 |
| 化神期 | 初/中/后/圆满，元神大成，神通广大 |
| 分神+合体 | 分神期分化元神 → 合体期身神合一 |
| 渡劫期 | 历经天劫，九死一生 |
| 大乘期 | 圆满飞升 |

### 三花聚顶体系
- **炼精化气**：藏精→养气→筑基→心动→玉液→结丹
- **炼气化神**：蕴丹→化元→玄光→外景→法相→凝神
- **炼神返虚**：阴神→神游→阳神→法身→人仙→地仙
- **炼虚合道**：天仙→玄仙→金仙→太乙→大罗→混元

## 二、玄幻升级流境界

### 全套境界链
```
玄徒→玄师→大玄师→玄灵→玄王→玄宗→玄尊→玄圣→玄皇→玄神→玄帝
```

### 变体
```
武者→玄者→玄师→玄灵→玄王→玄皇→玄宗→玄尊→玄圣→玄帝→散仙（九重）
```

### 神域/仙界进阶
```
天仙→太乙玄仙→金仙→大罗金仙→仙王→仙皇→仙帝→仙尊→主宰
天神→神王→神尊→神皇→神帝
```


## 五、物品/装备品阶

### 武器/防具
```
法器→法宝→灵器→灵宝→仙器→仙宝→神器（后天/先天）
```

变体：
```
符器→法器→纯阳→通天→神器→仙器→圣器→不朽圣器→玄天瑰宝→太初宝器→大道至宝
```

### 丹药
```
1-9阶（下/中/上/极品）→仙丹→神丹
```

常见丹药：筑基丹、养神丹（疗伤）、玄爆丹（临时提升）、隐身丹、万寿丹（增寿元）

## 六、修炼资质设定

```
废灵根→伪灵根→真灵根→天灵根→异灵根→仙灵根→混沌灵根
```

五行总量决定修行速度：五行越多→修行越快，资源消耗越少。
