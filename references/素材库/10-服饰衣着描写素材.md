# 服饰衣着描写素材

> 核心心法：服饰描写不是为了写衣服，而是为了写人——写身份、写性格、写情绪、写处境。好的服饰描写让读者"看见"角色，而不是"数衣服"。

---

## 一、服饰描写四技法

| 技法 | 操作 | 示例 |
|------|------|------|
| 以衣写人 | 用服饰折射性格/状态 | "他穿着一件洗得发白的衬衫，领口熨得一丝不苟——穷，但绝不邋遢。" |
| 以衣写情 | 用服饰传递情绪/关系 | "她穿着他的外套，袖子长出一截，她把手指缩在里面，贪恋那点温度。" |
| 以衣写境 | 用服饰暗示处境/阶层 | "在一屋子真丝旗袍里，她那件棉布连衣裙像误入了不属于她的空间。" |
| 留白写衣 | 不写全貌，只写冲击点 | "他从夜色里走出来，大衣领子竖着，只露出一双眼睛。" |

### 避坑铁律

> 不要罗列品牌、面料、款式如购物清单。每次最多写2-3个服饰特征，且必须与人物/情节相关。

## 六、现代服饰描写要点

### 用衣服写状态

| 状态 | 写法 |
|------|------|
| 精致 | "套装没有一丝褶皱，高跟鞋敲在地砖上的节奏像时钟一样精准。" |
| 落魄 | "西装还是那件好西装，但袖口磨出了线头，衬衫也有两天没换了。" |
| 匆忙 | "领带歪到了一边，衬衫下摆有一角没塞进裤腰。" |
| 强大 | "他站在那里，不需要名牌加持——存在感本身就是最贵的衣服。" |
| 防备 | "她穿着宽松的衣服，化着淡妆，把自己藏在一层'普通'的壳里。" |

### 氛围词汇库

| 风格 | 面料/质感 | 款式 | 颜色 | 氛围词 |
|------|----------|------|------|--------|
| 禁欲 | 羊绒/真丝/精纺羊毛 | 高领/收腰/长款 | 黑/深灰/藏蓝/白 | 矜贵/疏离/清冷 |
| 温润 | 棉麻/针织/灯芯绒 | 宽松/叠穿 | 米白/燕麦/驼色/浅灰 | 温暖/书卷气/干净 |
| 飒爽 | 皮/牛仔/精纺棉 | 收腰短款/阔腿裤 | 黑/酒红/军绿 | 利落/力量感/不羁 |
| 甜美 | 蕾丝/雪纺/薄纱 | 泡泡袖/A字裙 | 粉色/奶油色/碎花 | 软糯/天真/少女感 |

## 七、各题材服饰快查

| 题材 | 核心服饰 | 写作重点 |
|------|----------|----------|

## 八、服饰描写节奏

| 时机 | 写法 |
|------|------|
| 角色登场 | 2-3句写印象——风格/颜色/一处细节 |
| 场景转换 | 1句写变化——"她换了一条红裙子" |
| 关系转折 | 1-2句特写——TA注意到了某件衣服的细节 |
| 重大场合 | 3-5句精写——服饰辅助气场 |
| 日常场景 | 0句——不需要写，读者会自己脑补 |

> **铁律**：不要每章写衣服。只在服饰能传递信息的时候写。三个绝不写：日常不出彩的衣服不写、纯为凑字数的描写不写、与人物无关的服饰细节不写。
