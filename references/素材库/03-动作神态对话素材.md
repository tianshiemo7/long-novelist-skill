# 动作神态对话素材

> 告别"看了看""笑了笑""愣了愣"三板斧，让每个动作都有画面感。

---

## 一、"看"的N种写法

| 场景/情绪 | 替换方案 |
|----------|---------|
| 深情注视 | 凝眸注视、目光灼灼、视线无法移开、一眼望进去 |
| 警惕打量 | 冷眼打量、目光锐利、神念扫过、上下扫了一眼 |
| 不屑一顾 | 扫了一眼、目光轻描淡写地掠过、眼皮都懒得抬 |
| 专注观察 | 凝望、端详、审视、目光沉凝、一瞬不瞬地盯着 |
| 远处眺望 | 抬眼望向远方、极目远眺、目光穿过人群 |
| 偷看 | 余光瞥见、飞快地瞟了一眼、侧目偷觑 |
| 震惊地看 | 瞪大了眼、瞳孔骤然收缩、眼珠子快掉出来 |
| 迷茫地看 | 眼神涣散、目光空洞、视线没有焦点 |

**高级写法示例**：
- 他的目光在她脸上停了一瞬，又若无其事地移开了。
- 两人目光撞了一下，各自偏开头。

## 二、"笑"的N种写法

| 笑类型 | 描写方案 |
|--------|---------|
| 偷笑 | 噗嗤一声、嘴角偷偷往上翘、别过脸去捂嘴 |
| 冷笑 | 嘴角向上弯了弯，眼睛里却结着冰碴 / 一道极淡的弧度 |
| 苦笑 | 扯了扯嘴角，没笑出来 / 那笑比哭还难看 |
| 宠溺笑 | 眼底漾开一圈温柔、摇了摇头忍不住笑了 |
| 自嘲笑 | 嗤了一声，也不知道笑谁 / 嘴角的笑意带着涩 |
| 豪爽笑 | 仰头大笑、笑声朗朗/洪亮/震耳 |
| 意味深长 | 嘴角一勾，什么也没说 / 眼底掠过一丝不易察觉的笑意 |

**具体动作替代"笑了笑"**：
- 他摇了摇头，强忍住笑，唇角却止不住往上翘。
- 眼角挤出细纹。
- 喉间滚出一声低笑。

## 三、"愣/呆"的N种写法

| 场景 | 描写方案 |
|------|---------|
| 轻度愣住 | 动作微微一滞、手上的动作停了一拍 |
| 意外而愣 | 表情瞬间僵住、整个人像被按了暂停键 |
| 震惊而愣 | 嘴巴张了张又合上了、脑子嗡的一片空白 |
| 长时间呆立 | 像一座石雕、仿佛一个用尽力气的发条玩偶 |
| 回味后反应 | 像是把那句话咀嚼了良久、沉寂半晌才吐出几个字 |

## 四、对话标签系统

### 基础标签（可少量使用，约20%）
说、道、问、答、回、接

### 动作锚定（替代"说"，推荐）
将对话嵌入动作中，去掉标签：

> 他将头深深埋进掌心，声音从指缝间闷闷地透出来："我累了。"

> 林东猛地跨出一步，脚下青石板崩碎："你敢动她。"

### 神态呈现（替代"道"）
> 她嘴角向上弯了弯，眼睛里却结着冰碴："你真行啊。"

> 他目光像结了冰，一字一顿："你想好了再说。"

### 通用公式：表情 + 动作 + 语气 = 完整画面

| 维度 | 常用词汇 |
|------|---------|
| 表情 | 皱着眉、垂着眼、抿嘴、挑眉、眯眼、绷着脸 |
| 动作 | 往椅背上一靠、揉了揉眉心、绞着衣角、叩了叩桌面 |
| 语气 | 声音轻得像飘在风里、一字一顿、语气漫不经心、嗓音压得极低 |

## 五、常见肢体动作速查

| 想表达的 | 升级动作描写 |
|---------|-------------|
| 思考 | 指尖有一下没一下地敲着桌面、托着下巴目光放空 |
| 等待 | 在窗前来回踱步、反复掏出手机看时间、靠着墙数地上的瓷砖 |
| 不耐烦 | 脚尖有节奏地点着地面、手指敲击节奏越来越快 |
| 心虚 | 眼神飘忽始终不敢直视、手不自觉地摸后脑勺、说话时总清嗓子 |
| 得意 | 下巴微扬、脚步带风、哼着小调 |
| 失落 | 肩膀垮了下去、垂着头手指无意识地抠着裤子上的线头 |
| 坚定 | 咬紧后槽牙、目光沉凝直视前方、一把攥紧了拳 |
| 疲惫 | 揉了揉发胀的太阳穴、眼下一片青黑、身形微晃 |

## 六、避坑指南

| ❌ 错误 | ✅ 正确 |
|---------|--------|
| 每句对话都配动作 | 关键对话才重点写，过渡对话简洁 |
| 让粗人做文雅动作 | 动作必须符合人设（莽夫不踱步，书生不拍桌） |
| 完全不用"说/道" | 多人对话时保留少量基础标签避免混乱 |
| 同一动作反复出现 | 三页内主角笑5次→翻车，换用不同角度描 |
| 动作堆砌无目的 | 每个动作应有潜台词（紧张/心虚/掩饰） |
