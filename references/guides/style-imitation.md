# 风格模仿指南

## 联网搜索与分析流程

### 步骤1：搜索

当用户提供模仿对象（作家名/作品名），使用 `WebSearch` 执行以下搜索：

```
搜索1："{作家名} 写作风格 语言特点 分析"
搜索2："{作家名} 小说 写作技巧 叙事手法"
搜索3："{作品名} 文笔分析 经典段落"
搜索4："{作家名} 对话描写 人物塑造 特点"
```

### 步骤2：提取风格卡片

从搜索结果中提取以下维度：

#### 句式维度
- **句子长度偏好**：短句（10-20字）为主 / 中句（20-40字）为主 / 长句（40字+）为主 / 长短交替
- **句法复杂度**：简单句 / 复合句 / 排比句 / 对仗
- **段落长度**：短段落（2-3句）/ 中等（5-8句）/ 长段落（10句+）

#### 描写维度
- **描写密度**：高密度（大量细节）/ 中等 / 简练（只写关键）
- **感官偏好**：视觉为主 / 听觉为主 / 多感官并重
- **比喻频率**：频繁使用 / 偶尔使用 / 很少使用
- **环境描写**：情景交融 / 功能性地标 / 精简

#### 对话维度
- **口语化程度**：高度口语化 / 半书面半口语 / 偏书面
- **个性化程度**：每个角色有明显不同的说话风格 / 大致区分 / 风格统一
- **对话与描写的比例**：对话为主（60%+） / 均衡（40-60%） / 描写为主

#### 叙事维度
- **节奏**：快节奏（事件密集）/ 中等 / 慢热（大段铺垫）
- **信息密度**：高（每段都有新信息）/ 中等 / 低（注重氛围）
- **视角介入度**：叙述者存在感强（评论/抒情）/ 客观叙述 / 自由间接引语

#### 情感维度
- **情感表达方式**：直白 / 含蓄 / 通过动作暗示 / 通过对话传达
- **情感强度**：浓烈 / 克制 / 冷峻
- **幽默感**：有 / 偶尔 / 几乎没有

## 风格样本提取

如果搜索结果中包含代表性段落，提取并保存到 `06-风格设定.md`。

### 样本标注格式

```markdown
## 风格样本

### 样本1：[来源]
> [原文段落]
**风格要点**：[标注这个段落的句式、描写、对话特点]
**可模仿技法**：[具体可以怎么模仿]
```

## 模仿层级

不要追求100%模仿。分三个层级：

### 第一层：形似（表层）
- 句式长度接近
- 段落节奏接近
- 对话占比接近
- **可追求，通过调整实现**

### 第二层：神似（中层）
- 叙事视角的介入度
- 情感表达的含蓄/直白程度
- 描写的感官偏好
- **值得追求，但需要理解为什么**

### 第三层：魂似（深层）
- 作家的价值观和世界观
- 独特的幽默感或悲剧感
- 不可复制的人生经验
- **不必追求，保持自己的声音**

## 常见模仿对象的风格速写

### 网络文学流派

| 流派 | 句式特点 | 节奏 | 对话风格 |
|------|---------|------|---------|
| 升级流/爽文 | 短句为主，大量动作描写 | 极快 | 战斗对话多，口语化 |
| 凡人流 | 中等长度，细节丰富 | 中等偏慢 | 日常对话自然 |
| 赘婿/逆袭流 | 中短句，心理活动多 | 中等 | 打脸对话设计精巧 |
| 无限流 | 中句为主，设定说明多 | 快 | 策略讨论+吐槽 |
| 悬疑灵异 | 中短句，氛围描写多 | 由慢到快 | 对话暗藏信息 |

### 传统文学名家

| 作家 | 句式特点 | 叙事风格 |
|------|---------|---------|
| 金庸 | 中长句，古典白话 | 说书人视角，第三人称全知 |
| 古龙 | 极短句，分行多 | 冷峻克制，大量留白 |
| 刘慈欣 | 中等偏长，科普感 | 宏大叙事，技术细节信手拈来 |
| 余华 | 极简短句 | 冷静叙述残酷，情感克制 |
| 莫言 | 长句密集，通感丰富 | 魔幻现实，感官轰炸 |

> 注：以上为速写，实际模仿时仍需联网搜索获取更详细的分析。
