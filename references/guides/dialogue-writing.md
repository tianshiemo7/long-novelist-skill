# 对话写作规范

好对话揭示人物、推动情节、制造冲突。

---

## 三大核心原则

### 1. 对话必须有目的
每句对话至少完成以下之一：推动情节、揭示人物、制造冲突、传达信息、表达情感、制造悬念。

无效对话：
> "你好。""你好。""吃了吗？""吃了。"

### 2. 对话应该简洁
人们说话不写论文。
> "我想告诉你的是我认为我们应该立刻离开这里。" → "我们得马上走。"

### 3. 真实的人不会完整表达
真实对话充满打断、迟疑、话题转移、暗示而非明说。
> "我本来想告诉你，但是——""但是什么？""算了，没什么。"

---

## 潜台词

好的对话，真正含义在表面之下。

| 直接（乏味） | 潜台词（有趣） |
|-------------|---------------|
| "我很生气。" | "没事。我挺好的。真的。" |
| "我喜欢你。" | "你今天看起来...不错。" |
| "我不信任你。" | "谢谢你告诉我。我会记住的。" |

### 潜台词四种手法

**话题转移**：
> "你爱我吗？""你看了天气预报吗？明天有雨。"

**反问而非回答**：
> "你杀了他吗？""你觉得像我这样的人会做那种事？"

**谈论其他事物**：
> "你想我吗？""我妈昨天打电话来了。"

**沉默和动作**：
> "你愿意原谅我吗？" 她继续看杂志，翻了一页。

## 对话节奏

### 快节奏对话
短句、无标签、打断、快速升级。适用：争吵、追逐、紧急决策。

```
"左边！"
"来不及了——"
"跳！"
水面越来越近。
"抓紧！"
```

### 慢节奏对话
长句、停顿、迂回、潜台词丰富。适用：告白、试探、审讯、告别。

```
"这些年，"他端起茶杯，看着茶叶慢慢沉下去，"你有没有想过，如果当初——"

"没有。"她回答。太快了。

他放下茶杯。"我也是。"
沉默蔓延。窗外梧桐叶被风卷起又落下。
"其实，"她说，声音低了下去，"有时候会。"
```

### 沉默的戏剧力量
- **拒绝回答** → 暗示隐瞒或保护
- **反应延迟** → 暗示震惊或权衡
- **对话中断** → 暗示情感崩溃或决心
- **以动作代替回答** → 比语言更有力

```
"你是不是早就知道了？"
他不说话。拿起杯子喝水，放下，又拿起。
"我在问你话。"
他终于看向她。"你真想听答案？"
```

### 对话中的权力博弈

| 权力信号 | 高位角色 | 低位角色 |
|---------|---------|---------|
| 提问vs回答 | 提问（控制话题） | 被迫回答 |
| 打断vs被打断 | 随意打断 | 小心不被打断 |
| 话题控制 | 决定谈什么 | 被动跟随 |
| 沉默使用 | 用沉默施压 | 用沉默逃避 |

---

## 对话与动作结合

**一致（增强）**：
> "我爱你。"她紧紧抱住他，眼泪流下来。

**矛盾（揭示真相）**：
> "我完全支持你。"他目光看向别处，手在口袋里握紧拳头。

**动作替代标签**：
> "你在撒谎。"她拍案而起。
> "坐下。"他头也不抬。

## 对话场景速查

### 争吵 → 短句+打断+情绪升级
### 告白 → 迟疑+停顿+绕弯子
### 暧昧 → 双关语+试探+话里有话
### 告别 → 克制+未尽之言+表面平静下暗涌
### 审讯 → 权力不对等+施压+信息博弈

---

## 常见错误

1. **信息倾倒** — 角色互相说已知信息
2. **所有人说话一样** — 无法区分角色（每个角色应有独特用词、句式、语气、口头禅）
3. **过度礼貌** — 真实对话更粗糙
4. **无意义闲聊** — 除非有特殊目的
5. **说教** — 角色发表长篇哲学论述

---

## 长篇对话额外注意事项

1. **角色说话风格一致性**：每个角色的口头禅、说话节奏要在200章中保持一致。如果角色在第180章说话风格突变，需要是刻意的人物转变。
2. **对话中的信息节奏**：长篇不需要在一段对话中释放所有信息。可以分多次对话逐步释放。
3. **跨卷对话呼应**：第10章A对B说的某句话，在第200章可以被B原封不动还回去——形成镜像和回响。
4. **对话中的伏笔**：对话是埋设伏笔最自然的方式。角色随口说的一句话，100章后可能是关键线索。
5. **方言和口癖管理**：记录在 `03-角色档案/角色/{角色名}/角色信息.md` 的 `speechStyle` 字段中，每次写作前回顾。
