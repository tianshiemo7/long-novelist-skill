# 情节结构模板

> 常见的故事结构模板。百万字长篇需要多层结构嵌套：全书有总结构，每卷有卷结构，每章有章结构。

---

## 一、起承转合：百万字长篇总结构

长篇小说的铁律结构，是其他所有模板的顶层框架。

| 阶段 | 占比 | 字数 | 核心任务 |
|------|------|------|---------|
| **起** | 20% | 前20万字 | 建立世界观、引入主角、铺设核心冲突 |
| **承** | 30% | 20-50万字 | 冲突升级、人物成长、支线展开 |
| **转** | 30% | 50-80万字 | 重大转折、真相揭露、人物蜕变 |
| **合** | 20% | 80-100万字 | 最终对决、伏笔回收、结局收束 |

每卷内部也应包含小型起承转合。

---

## 二、三幕式结构（Three-Act Structure）

最经典的故事结构，适用于大多数长篇小说。

### 第一幕：设置 — 约25%

| 章节 | 内容 |
|-----|------|
| 第1-N章 | 介绍主角日常生活（现状） |
| 激励事件 | 打破现状的事件 |
| 第一幕结尾 | 主角拒绝召唤，但最终决定行动 |

**第一幕悬念示例：** "门开了，进来的人是三年前死去的人。"

### 第二幕：对抗 — 约50%

| 章节 | 内容 |
|-----|------|
| 前半段 | 主角进入新世界，遭遇挑战 |
| 中点 | 重大转折/信息揭示 |
| 后半段 | 困难升级，盟友可能背叛 |

**第二幕悬念示例：** "导师竟然是幕后黑手。"

### 第三幕：结局 — 约25%

| 章节 | 内容 |
|-----|------|
| 前段 | 一切看似失败，最低点 |
| 高潮 | 最终对决 |
| 结尾 | 结局，展示新常态 |

---

## 三、英雄之旅（Hero's Journey）

神话学家约瑟夫·坎贝尔的经典结构，适合冒险/奇幻/玄幻题材。

### 十二阶段分解（百万字适配）

| 阶段 | 字数位置 | 内容 |
|-----|---------|------|
| 1. 平凡世界 | 前3-5万字 | 介绍主角日常，埋下"不凡"的种子 |
| 2. 冒险召唤 | 3-8万字 | 激励事件发生 |
| 3. 拒绝召唤 | 5-10万字 | 主角最初犹豫 |
| 4. 遇见导师 | 8-15万字 | 获得指导/装备/金手指 |
| 5. 跨越门槛 | 10-20万字 | 离开舒适区，进入新世界 |
| 6. 考验盟友敌人 | 15-30万字 | 新世界探索，建立关系网 |
| 7. 接近洞穴 | 25-40万字 | 准备面对大挑战 |
| 8. 苦难煎熬 | 35-50万字 | 接近死亡/重大失败 |
| 9. 奖赏 | 45-60万字 | 获得力量/信息/突破 |
| 10. 返回之路 | 55-75万字 | 回归途中受阻 |
| 11. 复活 | 70-90万字 | 最终考验/蜕变 |
| 12. 满载而归 | 85-100万字 | 回归，带着收获 |

## 四、悬疑小说结构

### 第一幕：谜题出现

| 章节 | 内容 | 悬念钩子 |
|-----|------|---------|
| 第1-N章 | 发现尸体/事件发生 | 谁干的？ |
| 后续 | 侦探接手案件 | 为什么这个案子特殊？ |
| 后续 | 初步调查，发现线索 | 线索指向谁？ |

### 第二幕：调查深入

| 章节 | 内容 | 悬念钩子 |
|-----|------|---------|
| 推进 | 审讯嫌疑人，各有嫌疑 | 谁在撒谎？ |
| 转折 | 新线索出现，指向意外方向 | 我们是不是一开始就错了？ |
| 升级 | 第二起事件，模式浮现 | 这是连环案件？ |
| 危机 | 侦探陷入危险 | 侦探会成为下一个目标吗？ |

### 第三幕：真相揭示

| 章节 | 内容 | 悬念钩子 |
|-----|------|---------|
| 突破 | 重大突破/反转 | 我们信任的人有问题？ |
| 对决 | 最终对决 | 真相是什么？代价是什么？ |
| 收尾 | 案件解决，遗留疑问 | 真的结束了吗？ |

## 五、言情小说结构

### 第一幕：相遇

| 章节 | 内容 | 情感节点 |
|-----|------|---------|
| 引入A | 介绍主角A，现状/问题 | 读者共情A |
| 引入B | 介绍主角B，现状/问题 | 读者共情B |
| 相遇 | A和B相遇（第一印象不佳） | 两人注定在一起，但目前有冲突 |

### 第二幕：发展

| 章节 | 内容 | 情感节点 |
|-----|------|---------|
| 靠近 | 被迫在一起，了解对方 | 发现有吸引力的地方 |
| 升温 | 吸引力增强，亲密时刻 | 接近 |
| 波折 | 误解/秘密/障碍出现 | 推远 |
| 危机 | 危机，关系破裂 | 看起来无望 |

### 第三幕：和解

| 章节 | 内容 | 情感节点 |
|-----|------|---------|
| 醒悟 | 意识到真爱，克服障碍 | 高潮 |
| 结局 | 在一起，展示新生活 | 圆满 |

## 六、反转结构（Twist-Based）

适合心理惊悚/悬疑/反套路文。

### 章节分布（长篇适配）

```
第1-30章：建立"真实"情况
第31-60章：出现疑点，但不明显
第61-90章：第一次重大反转（重新理解一切）
第91-150章：建立新理解，继续深入
第151-200章：第二次反转（再次颠覆）
第201-250章：第三次反转（一切颠覆）
第251-300章：最终收束
```

### 反转三原则

1. 前期埋下看似无辜的线索
2. 每次反转回看都有迹可循
3. 避免机械降神

---

## 七、多线叙事结构

适合复杂剧情，多主角/多视角。

### 交叉剪辑模式（长篇）

```
第1-5章：主角A故事线
第6-10章：主角B故事线
第11-13章：主角A故事线（推进）
第14-16章：主角B故事线（推进）
第17-20章：线索交汇
...每卷至少交汇1-2次
```

### 时间线模式

```
第1-N章：现在（时间线A）
穿插章节：过去（时间线A-若干年）
交替推进，最终两条线在某关键节点汇合
```

### 收敛原则

- 各线最终必须交汇
- 早期看似无关的事件后来有关联
- 收敛时产生"原来如此"的效果
- 百万字长篇中，多线交汇点至少设置3-5个，分布在各卷结尾

---

## 八、单章内部结构

```
开头（10%）：钩子 + 上下文连接 → 制造好奇心
发展（60%）：事件推进 + 冲突 → 推动剧情
高潮（20%）：本章最高点 → 满足读者预期
结尾（10%）：悬念钩子 + 下章铺垫 → 让读者点下一章
```

### 单章必须包含的元素
- 至少一个冲突点（人物/内心/环境冲突）
- 至少一个剧情推进（不能原地踏步）
- 至少一个情感锚点（让读者与角色共情）
- 一个卡点/钩子结尾

---

## 九、结构选择指南

| 故事类型 | 推荐主结构 | 推荐子结构 |
|---------|-----------|-----------|
| 玄幻/仙侠 | 英雄之旅 + 起承转合 | 多线叙事 |
| 悬疑/推理 | 悬疑结构 + 反转结构 | 多线叙事 |
| 言情/甜宠 | 言情结构 | 起承转合 |
| 科幻/末世 | 三幕式 + 起承转合 | 多线/反转 |
| 历史/权谋 | 起承转合 + 多线叙事 | 反转结构 |
| 无限流 | 英雄之旅（每卷）+ 副本结构 | 多线/反转 |

---

## 十、无效章节检测

以下情况说明章节结构有问题：

- 开头：漫长铺垫/背景说明（→ 用钩子开场）
- 中间：日常活动/对话流水账（→ 加入冲突）
- 结尾：平淡结束/无悬念（→ 用钩子收尾）
- 整章：主角没有推动剧情（→ 让主角做决定）
- 整章：事件推进为零（→ 删或合并到其他章）
