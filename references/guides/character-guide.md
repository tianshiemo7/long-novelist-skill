# 人物塑造指南

好的人物是故事的灵魂。读者记住的是人，不是情节。

---

## 核心原则：矛盾创造深度

真实的人充满矛盾。矛盾是人物的发动机。

**有效矛盾示例：**
- 雷厉风行的女警长，回家后对着亡夫的照片说话
- 满口仁义的书生，暗中替人写休书赚银子
- 冷血杀手，每天给流浪猫带鱼
- 最不信命的人，抽屉里藏着一张求来的护身符

**无效矛盾**：为矛盾而矛盾，与人物核心无关。

---

## 侧面揭示技法

不要直接陈述性格，通过行为展示。

| 直接陈述（弱） | 侧面展示（强） |
|---------------|---------------|
| 他很愤怒 | 捏碎手中纸杯，碎片扎进掌心也没松手 |
| 她很紧张 | 反复调整眼镜位置，第三次了 |
| 他很傲慢 | 从不直视下属眼睛，对话时总看窗外 |
| 她很善良 | 偷偷喂了三年流浪狗，谁也没告诉 |
| 他很孤独 | 电视永远开着，他从来不看 |

### 用选择揭示本质

人物在压力下的选择，比任何描述都有说服力。

> 火灾现场，他可以救名画，也可以救那只猫。他抱起了猫。
> 不需要写"他善良"，这个选择已经说了。

### 用习惯暴露真相

反复出现的小习惯是人物的无意识自我暴露：
- 紧张时摸无名指（曾经戴过戒指？）
- 总坐背靠墙的位置（缺乏安全感？）
- 说话时避开眼神接触（隐瞒什么？）

## 主角塑造

### 主角必须有
- **明确目标** — 他想要什么
- **强大动机** — 为什么想要
- **可共情性** — 读者能理解他的感受
- **成长空间** — 故事中会改变

### 主角原型

| 类型 | 特征 | 故事作用 |
|-----|------|---------|
| 英雄型 | 勇敢、正义、利他 | 战胜外在威胁 |
| 成长型 | 从弱小到强大 | 克服内在缺陷 |
| 反英雄型 | 道德灰色、复杂 | 挑战传统道德 |
| 平凡型 | 普通人卷入非凡事 | 读者代入感强 |

### 缺陷致命化

主角必须有缺陷，缺陷在关键时刻导致失败。没有缺陷的角色不会成长。

| 缺陷 | 导致的失败 | 成长方向 |
|-----|----------|---------|
| 傲慢 | 低估对手，落入陷阱 | 学会倾听 |
| 信任问题 | 拒绝帮助，孤立无援 | 学会信任 |
| 完美主义 | 无法及时行动，错失机会 | 接受不完美 |
| 复仇心 | 被利用，失去理智 | 学会放下 |
| 逃避 | 问题越滚越大 | 学会面对 |

## 反派塑造

好反派不是坏人，是**信念不同的人**。

### 好反派的标准
- **强大可信** — 不应该是草台班子
- **有自己的逻辑** — 相信自己在做正确的事
- **与主角有深层联系** — 不是单纯为作恶
- **揭示主题** — 挑战主角的信念

### 反派动机层次

**浅层**（不够）："我想毁灭世界" → 乏味

**中层**（及格）："我失去了一切，世界对我无意义" → 单薄

**深层**（有力）："女儿死于可治愈的病，药厂为利润延迟了上市。这个世界不值得被拯救。" → 读者虽不认同但能理解

### 反派与主角的镜像关系

最好的反派是主角的**黑暗镜像**——如果主角走了另一条路就会变成他。

> 主角：失去家人后选择保护他人
> 反派：失去家人后选择毁灭一切
> 同一起点，不同选择。

## 配角的功能性

配角存在的唯一理由：**对主角产生作用**。

| 类型 | 功能 | 注意 |
|-----|------|------|
| 导师型 | 指引主角，传递信息 | 必须有自己盲区 |
| 盟友型 | 协助主角，提供支持 | 不能只是工具人 |
| 搞笑型 | 缓解紧张 | 关键时刻也能严肃 |
| 叛徒型 | 制造转折 | 背叛要有前期铺垫 |

### 配角原则
- 删除"只是存在"的角色
- 每个配角有明确作用
- 避免刻板印象（除非刻意为之）

## 人物关系设计

### 关系动态变化

```
第一章：陌生人
第三章：不情愿的盟友
第五章：建立信任
第七章：背叛/考验
终章：真正的友谊（或决裂）
```

### 关系揭示层次
- **表层**：表面互动
- **中层**：共同经历后的默契
- **深层**：真实感受/秘密

### 高戏剧潜力关系

| 关系类型 | 戏剧潜力 | 经典模式 |
|---------|---------|---------|
| 亦敌亦友 | 高 | "我知道你在利用我，但我需要你" |
| 禁忌之爱 | 高 | 身份/立场对立，无法控制感情 |
| 师徒 | 中 | "教会徒弟饿死师父"的潜在张力 |
| 兄弟竞争 | 中 | 同一起点不同选择，此消彼长 |

## 人物出场设计

### 有效出场方式
- **行动中** — 展示能力或性格
- **冲突中** — 立即建立关系/对立
- **误解中** — 建立悬念

| 无效出场 | 有效出场 |
|---------|---------|
| 李明，28岁，是一名侦探。他走进办公室。 | 李明跨过警戒线，警官试图拦住他。"市刑警队，李明。"他亮出证件，径直走向尸体。 |
| 美丽的女孩坐在窗边，她叫小红。 | 她已经三天没睡了，咖啡杯里的液体在颤抖。门铃响起时，她几乎把杯子摔在地上。 |

**禁止**：镜子自照描写外貌、简历式介绍、无意义日常活动。

---

## 长篇人物管理要点

1. **分批登场**：不要前10章把所有角色都推出来。按卷引入新角色。
2. **退场机制**：配角该退场就退场（死亡/离开/任务完成），不要600章后还有50个"活跃"角色。
3. **状态追踪**：`03-角色档案/角色/{角色名}/角色信息.md` 中的 `status` 和 `currentLocation` 每10章至少更新一次。
4. **关系动态变化**：每20章检查一次角色间的关系是否有推进，不能200章后还是"亦敌亦友"原地踏步。
5. **说话风格一致性**：每个角色的 `speechStyle` 要在全书中保持一致（除非是刻意转变）。
